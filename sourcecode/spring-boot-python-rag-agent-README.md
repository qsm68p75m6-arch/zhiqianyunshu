# Spring Boot + Python RAG Agent 项目

## 项目概述
这是一个基于Spring Boot后端和Python AI服务的RAG（检索增强生成）系统，集成了Agent能力。

## 文件夹结构说明

### `backend/` - Spring Boot后端服务
- 包含Java代码、配置文件、API接口
- 负责业务逻辑、数据访问、缓存管理
- 前端API网关

### `ai-service/` - Python AI服务
- 基于FastAPI的AI微服务
- RAG核心引擎
- Embedding模型、向量数据库、LLM集成
- Agent推理服务

### `docs/` - 项目文档
- 架构设计文档
- API接口文档
- 部署指南
- 学习笔记
